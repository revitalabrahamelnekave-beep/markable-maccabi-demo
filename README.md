# PERI-AGENT 🌸 — סוכן קידום SEO / GEO ל-MARKABLE

סוכן Python אוטונומי שמקדם את **MARKABLE (Peritale)** במנועי חיפוש ובמנועי AI —
**בבטחה מלאה ובעלות נמוכה**. הסוכן מבצע **קריאה וניתוח בלבד**: הוא לעולם לא מגיש
טפסים, לא מפרסם תוכן, ולא מבצע שום פעולה שיוצאת החוצה. כל פלט הוא **דוח או טיוטה**
שאתה בודק ומפעיל ידנית.

> למה זה בטוח? כי הסיכון האמיתי ב-SEO ב-2026 הוא **ספאם** (עדכוני הספאם של גוגל
> תוקפים דפוסי לינקים מניפולטיביים). הסוכן הזה לא מייצר ספאם — הוא מנתח, ממליץ,
> ומכין טקסטים איכותיים שאתה מדביק בעצמך היכן שתבחר.

## מה הסוכן עושה (לפי סדר אפקטיביות)

| # | משימה | מה זה נותן | עלות רשת |
|---|---|---|---|
| **1** | **אודיט ציטוטים ב-AI (GEO)** ⭐ | שואל את השאלות שהקהל שלך שואל את ChatGPT/Perplexity (עם חיפוש רשת חי) ובודק אם Markable מצוטט מול מתחרים. זה השדה הכי חשוב ב-2026. | גבוהה (חיפושי רשת) |
| **2** | **אודיט GEO/SEO של עמודים** | מוריד את עמודי האתר ובודק TL;DR, מבנה שאלה-תשובה, סכמה, meta, טבלאות, alt — ומחזיר המלצות ממופות לפי עדיפות. | נמוכה |
| **3** | **מילות מפתח ופערי תוכן** | מייצר 12–15 שאילתות high-intent (עברית+אנגלית) עם רעיון תוכן והערת GEO לכל אחת. | נמוכה |
| **4** | **ערכת הגשה לדיירקטוריז** | רשימת דיירקטוריז איכותיים רלוונטיים + טקסטים מוכנים להדבקה (עברית+אנגלית). אתה מגיש ידנית. | נמוכה |

הכל נאסף ל**דוח Markdown אחד** עם חותמת זמן בתיקיית `reports/`.

## התקנה

```bash
pip install -r requirements.txt
```

הגדר מפתח API (או השתמש ב-`ant auth login`):

```bash
cp .env.example .env      # וערוך, או פשוט:
export ANTHROPIC_API_KEY=sk-ant-...
```

## הרצה

```bash
# הצג תוכנית + הערכת עלות, בלי לקרוא ל-API:
python -m peri_agent.main --dry-run

# הרץ הכל:
python -m peri_agent.main

# הרץ משימה אחת בלבד:
python -m peri_agent.main --only geo
python -m peri_agent.main --only pages
python -m peri_agent.main --only keywords
python -m peri_agent.main --only directories
```

בסוף כל ריצה מודפסת **עלות משוערת** בדולרים.

---

## עלות ובקרת עלות 💰

הכל נשלט מ-`config.yaml`. מנופי העלות העיקריים:

1. **מודל** — ברירת המחדל `claude-opus-4-8` (החזק ביותר). **להוזלה משמעותית** החלף ל:
   - `claude-haiku-4-5` — פי ~5 זול יותר, מספיק לרוב המשימות
   - `claude-sonnet-5` — פי ~1.6 זול מ-opus, איכות כמעט זהה
2. **מספר שאלות GEO** (`geo_queries`) — כל שאלה = חיפוש רשת (מנוע העלות העיקרי). קצץ את הרשימה כדי להוזיל.
3. **`max_web_searches`** — מקס' חיפושים לכל שאלה.
4. **`effort`** — `low` זול ומהיר יותר מ-`high`.

הרצה מלאה טיפוסית עם ברירות המחדל היא בסדר גודל של כמה עשרות סנט עד ~2$. הרץ
`--dry-run` תמיד קודם.

---

## הגדרה (`config.yaml`)

זה הקובץ היחיד שצריך לערוך:

- `product` — פרטי המוצר (כבר מלא ל-Markable/Peritale)
- `geo_queries` — השאלות לאודיט ה-GEO (ערוך למה שהקהל שלך באמת שואל)
- `competitors` — מתחרים לבדיקה
- `pages_to_audit` — כתובות עמודים לאודיט
- `directories` / `keywords` — הפעלה/כיבוי
- `llm` — מודל, effort, מקס' חיפושים
